# Анализ финансовых транзакций и клиентов

### 1. Очистка и подготовка данных
- Удаление строк с пропущенными `transaction_id`, `client_id`, `transaction_date`
- Приведение дат к стандартному формату
- Удаление отрицательных сумм транзакций
- Фильтрация по услугам:
  - Инвестиционное консультирование
  - Налоговое планирование
  - Структурирование капитала
  - Управление активами
  - Финансовое планирование
- Удаление записей с неизвестными городами и способами оплаты

### 2. Анализ данных транзакций
- Статистика по суммам транзакций
- Анализ по типам услуг
- Анализ по городам и способам оплаты

### 3. Объединение данных и анализ по уровням активов
- Объединение транзакций с данными клиентов по `client_id`
- Категоризация клиентов по уровню активов:
  - **Низкий капитал** (< 100 000)
  - **Средний капитал** (100 000 – 1 000 000)
  - **Высокий капитал** (> 1 000 000)
- Анализ выручки по категориям

### 4. Визуализация данных
- Распределение сумм транзакций
- Выручка по услугам
- Зависимость средней суммы транзакции от возраста
- Сохранение графиков в папку `visualizations/`

## Требования

- Python 3.12+

## Запуск

```bash
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
python3 main.py
```
